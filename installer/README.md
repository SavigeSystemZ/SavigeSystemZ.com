# Installer and Packaging

## Linux install paths
- `.deb` package under `installer/packaging/deb`
- AppImage configuration under `installer/packaging/appimage`
- Desktop launcher template in `installer/savigesystemz.desktop`

## Scripts
- `installer/scripts/install.sh` - local install and desktop integration
- `installer/scripts/uninstall.sh` - remove launcher and assets
- `installer/scripts/validate-install.sh` - post-install checks
- `installer/scripts/build-packages.sh` - package build scaffold
